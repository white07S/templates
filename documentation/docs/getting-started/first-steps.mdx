# First Steps

Let's create your first project and explore the basic concepts of our platform.

## Creating Your First Project

### Step 1: Initialize a New Project

Open your terminal and run:

```bash
# Create a new project
our-product init my-first-app

# Navigate to the project directory
cd my-first-app

# Install dependencies
npm install
```

### Step 2: Project Structure

After initialization, your project structure will look like this:

```
my-first-app/
├── src/
│   ├── components/
│   ├── pages/
│   ├── styles/
│   └── index.js
├── public/
│   └── index.html
├── config.json
├── package.json
└── README.md
```

### Step 3: Start the Development Server

```bash
# Start the development server
npm start

# Your app is now running at http://localhost:3000
```

## Understanding the Basics

### Components

Components are the building blocks of your application:

```javascript
// src/components/HelloWorld.js
import React from 'react';

function HelloWorld({ name }) {
  return (
    <div className="greeting">
      <h1>Hello, {name}!</h1>
      <p>Welcome to your first component.</p>
    </div>
  );
}

export default HelloWorld;
```

### Pages

Pages represent different routes in your application:

```javascript
// src/pages/Home.js
import React from 'react';
import HelloWorld from '../components/HelloWorld';

function HomePage() {
  return (
    <div className="home-page">
      <HelloWorld name="Developer" />
      <p>This is your home page.</p>
    </div>
  );
}

export default HomePage;
```

### Routing

Set up basic routing:

```javascript
// src/App.js
import { BrowserRouter, Routes, Route } from 'react-router-dom';
import HomePage from './pages/Home';
import AboutPage from './pages/About';

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path="/" element={<HomePage />} />
        <Route path="/about" element={<AboutPage />} />
      </Routes>
    </BrowserRouter>
  );
}
```

## Your First Feature

Let's add a simple counter feature:

### Create the Counter Component

```javascript
// src/components/Counter.js
import React, { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);

  const increment = () => setCount(count + 1);
  const decrement = () => setCount(count - 1);
  const reset = () => setCount(0);

  return (
    <div className="counter">
      <h2>Counter: {count}</h2>
      <div className="button-group">
        <button onClick={decrement}>-</button>
        <button onClick={reset}>Reset</button>
        <button onClick={increment}>+</button>
      </div>
    </div>
  );
}

export default Counter;
```

## Working with Data

### Fetching Data

Here's how to fetch data from an API:

```javascript
// src/services/api.js
export async function fetchUsers() {
  try {
    const response = await fetch('https://jsonplaceholder.typicode.com/users');
    const data = await response.json();
    return data;
  } catch (error) {
    console.error('Error fetching users:', error);
    return [];
  }
}
```

### Using the Data

```javascript
// src/components/UserList.js
import React, { useState, useEffect } from 'react';
import { fetchUsers } from '../services/api';

function UserList() {
  const [users, setUsers] = useState([]);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    async function loadUsers() {
      const data = await fetchUsers();
      setUsers(data);
      setLoading(false);
    }
    loadUsers();
  }, []);

  if (loading) return <div>Loading users...</div>;

  return (
    <div className="user-list">
      <h2>Users</h2>
      <ul>
        {users.map(user => (
          <li key={user.id}>
            {user.name} - {user.email}
          </li>
        ))}
      </ul>
    </div>
  );
}

export default UserList;
```

## State Management

For complex state management, use our built-in store:

```javascript
// src/store/index.js
import { createStore } from '@our-product/store';

const initialState = {
  user: null,
  theme: 'light',
  notifications: []
};

export const store = createStore(initialState);

// Actions
export const setUser = (user) => {
  store.setState({ user });
};

export const toggleTheme = () => {
  const currentTheme = store.getState().theme;
  store.setState({
    theme: currentTheme === 'light' ? 'dark' : 'light'
  });
};
```

## Best Practices

1. **Keep Components Small**: Each component should have a single responsibility
2. **Use Meaningful Names**: Choose descriptive names for variables and functions
3. **Write Tests**: Add tests for critical functionality
4. **Optimize Performance**: Use lazy loading and code splitting
5. **Handle Errors**: Always include error handling in your code

## Common Patterns

### Loading States

```javascript
function DataComponent() {
  const [data, setData] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  // Show loading
  if (loading) return <LoadingSpinner />;

  // Show error
  if (error) return <ErrorMessage error={error} />;

  // Show data
  return <DataDisplay data={data} />;
}
```

### Form Handling

```javascript
function ContactForm() {
  const [formData, setFormData] = useState({
    name: '',
    email: '',
    message: ''
  });

  const handleChange = (e) => {
    setFormData({
      ...formData,
      [e.target.name]: e.target.value
    });
  };

  const handleSubmit = (e) => {
    e.preventDefault();
    // Process form data
    console.log('Submitted:', formData);
  };

  return (
    <form onSubmit={handleSubmit}>
      <input
        name="name"
        value={formData.name}
        onChange={handleChange}
        placeholder="Name"
      />
      <input
        name="email"
        type="email"
        value={formData.email}
        onChange={handleChange}
        placeholder="Email"
      />
      <textarea
        name="message"
        value={formData.message}
        onChange={handleChange}
        placeholder="Message"
      />
      <button type="submit">Send</button>
    </form>
  );
}
```
