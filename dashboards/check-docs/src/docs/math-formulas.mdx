This page demonstrates mathematical formula rendering and code blocks with copy functionality.

## Mathematical Formulas

### Inline Math

You can include inline math formulas like $E = mc^2$ or the quadratic formula $x = \frac\{-b \pm \sqrt\{b^2-4ac\}\}\{2a\}$ directly in your text.

### Block Math Formulas

For more complex mathematical expressions, use block math:

```math
\int_{-\infty}^{\infty} e^{-x^2} dx = \sqrt{\pi}
```

### Advanced Mathematical Examples

#### Calculus

The fundamental theorem of calculus:

```math
\int_a^b f'(x) \, dx = f(b) - f(a)
```

#### Linear Algebra

Matrix multiplication:

```math
\begin{pmatrix}
a & b \\
c & d
\end{pmatrix}
\begin{pmatrix}
e & f \\
g & h
\end{pmatrix}
=
\begin{pmatrix}
ae + bg & af + bh \\
ce + dg & cf + dh
\end{pmatrix}
```

#### Statistics

Normal distribution probability density function:

```math
f(x) = \frac{1}{\sigma\sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x-\mu}{\sigma}\right)^2}
```

#### Complex Analysis

Euler's formula:

```math
e^{i\theta} = \cos\theta + i\sin\theta
```

#### Differential Equations

The general solution to a second-order linear homogeneous equation:

```math
y'' + p(x)y' + q(x)y = 0
```

### JavaScript

```javascript
// Fibonacci sequence generator
function fibonacci(n) {
  if (n <= 1) return n;
  return fibonacci(n - 1) + fibonacci(n - 2);
}

// Memoized version for better performance
const fibMemo = (function() {
  const cache = {};
  return function fib(n) {
    if (n in cache) return cache[n];
    if (n <= 1) return n;
    return cache[n] = fib(n - 1) + fib(n - 2);
  };
})();

console.log(fibonacci(10)); // 55
console.log(fibMemo(10)); // 55
```

-- Complex query with window functions
SELECT
    employee_id,

    RANK() OVER (PARTITION BY department_id ORDER BY salary DESC) as salary_rank,

### Bash/Shell

```bash
#!/bin/bash

# Automated backup script
BACKUP_DIR="/backup"
SOURCE_DIR="/home/user/documents"
DATE=$(date +%Y%m%d_%H%M%S)
BACKUP_FILE="backup_$DATE.tar.gz"

# Create backup directory if it doesn't exist
mkdir -p "$BACKUP_DIR"

# Create compressed backup
echo "Creating backup: $BACKUP_FILE"
tar -czf "$BACKUP_DIR/$BACKUP_FILE" "$SOURCE_DIR"

# Check if backup was successful
if [ $? -eq 0 ]; then
    echo "Backup completed successfully!"

    # Remove backups older than 30 days
    find "$BACKUP_DIR" -name "backup_*.tar.gz" -mtime +30 -delete
    echo "Old backups cleaned up."
else
    echo "Backup failed!"
    exit 1
fi
```

- **Inline Math**: Use single dollar signs for inline formulas like $\alpha + \beta$

- **Responsive**: Math formulas adapt to different screen sizes
